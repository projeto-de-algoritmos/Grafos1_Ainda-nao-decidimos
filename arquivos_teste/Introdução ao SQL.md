# Introdução ao SQL
Sql (**S**tructured **Q**uery **L**anguage) é uma linguagem de programção cujo principal propósito é gerenciar dados armazenados em [[Banco de dados Relacionais]]. Busca operar e, busca operar com simples declarações, a fim de manter a precisão e segurança, mantendo a integridade dos dados que opera, independente de sua magnitude.

O Sql segue uma estrutura principal ao redor de [[Declaracoes]]

Acesse aqui a [Lista](https://www.codecademy.com/article/sql-commands) com todos os comandos SQL.
